# Apex-Framework-BTA

### 什么是Apex-Framework-BTA?

Apex-Framework-BTA是由波西米亚旅游协会成员共同协助和开发的Apex-Framework的一个分支，用于旅游团所属的公开服务器。

### 主要内容

Invade & Annex : Apex (原版-主分支)

Invade & Annex : Opposing force (CSAT版本-dev分支)

Invade & Annex : RHS (RHS版本-dev分支)
